# 🌌 Henox - Premium Discord & Spotify Portfolyo

Gerçek zamanlı Discord entegrasyonu, yüzen bir Spotify oynatıcısı ve modern "glassmorphism" tasarımıyla öne çıkan, yüksek estetiğe sahip etkileşimli bir portfolyo sitesi. 

## ✨ Temel Özellikler

- **🛡️ Discord Kimliği**: Lanyard API kullanarak Discord kullanıcı adınızın, avatarınızın ve durumunuzun gerçek zamanlı gösterimi.
- **🏅 Etkileşimli Rozetler**: Discord rozetlerinizi hızlı ve şık bir özel tooltip sistemiyle sergiler.
- **🎶 Yüzen Spotify Kartı**: Spotify'da müzik dinlerken gerçek zamanlı güncellenen, ekranın altında süzülen bağımsız bir müzik çalar.
- **🎞️ Dinamik Banner**: Profil kartının üst kısmında yüksek kaliteli video veya resim banner desteği.
- **✨ Premium Animasyonlar**: GSAP destekli akıcı açılış animasyonları ve mouse hareketlerine duyarlı tilt efekti.
- **📺 Sinematik Arkaplan**: Derin ve atmosferik bir deneyim için tarama çizgileri, noise efekti ve döngüsel video arkaplanı.

## 🚀 Başlangıç

1. Depoyu (repository) klonlayın veya indirin.
2. `index.html` dosyasını tarayıcınızda açın (Live Server kullanılması önerilir).
3. Discord verilerini kendinize göre ayarlamak için:
   - `script.js` dosyasını açın.
   - `DISCORD_USER_ID` kısmını kendi ID'nizle değiştirin.
